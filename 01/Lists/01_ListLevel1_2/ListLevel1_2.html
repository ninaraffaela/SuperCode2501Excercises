Aufgabenstellung

  • Lernziel: Das Erstellen eines Inhaltsverzeichnisses unter Verwendung von unordered list und ordered list.
  • Ein klassisches Beispiel für Listen ist das Inhaltsverzeichnis. Es kann ganz einfach strukturiert sein, manchmal aber eben auch komplexer, d.h. das Inhaltsverzeichnis ist in verschiedene Ebenen verschachtelt.
  • Stell dir vor du schreibst eine Masterarbeit, ein Buch o.Ä. Bitte erstelle dazu ein Inhaltsverzeichnis.
  • Schau dir die Ergebnisvorschau an.

    Hinweise:

  • Nutze das HTML-Element unordered list "ul".
  • Nutze das HTML-Element unordered list "ol".
  • Vergiss nicht deinen Code einzurücken, so bleibt es übersichtlicher!
